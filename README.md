# Drveće Banjaluke

https://banjalukatrees.pages.dev/

A tree map of Banjaluka,
Includes:

- `schema.sql` — Postgres/PostGIS schema
- `index.html` — a working single-file html (map + tree detail panel +
  3D model viewer)

## 1. Backend (Supabase)

1. A project at supabase.com.
2. SQL `schema.sql` creates the tables,
   PostGIS geography column, row-level security policies, and seeds data.
## 2. Frontend
`index.html` to real data
